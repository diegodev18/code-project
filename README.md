# Code Project - Aprende programacion con proyectos reales!

## Descripcion

Este proyecto es una recopilacion de proyectos de programacion para aprender a programar de forma practica y divertida. Cada proyecto tiene una descripcion, una lista de materiales necesarios, un diagrama de conexionado y el codigo fuente.

## Contribuir

Si quieres contribuir a este proyecto, puedes hacerlo de las siguientes formas:

- **Añadir un nuevo proyecto**: Si tienes un proyecto que quieras compartir, puedes añadirlo a la lista de proyectos. Para ello, crea un nuevo archivo markdown en la carpeta `projects` con la informacion del proyecto y el codigo fuente.

- **Corregir errores**: Si encuentras algun error en algun proyecto, puedes corregirlo y enviar un pull request, o enviar un issue.

- **Mejorar la documentacion**: Si crees que la documentacion de algun proyecto se puede mejorar, puedes hacerlo y enviar un pull request.
